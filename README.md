# NippoCore
日報システム **(製作中)**

## Usage
メインのシステムにマウントしてください

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'nippo_core'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install nippo_core
```
